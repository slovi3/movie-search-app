# Movie Search App

Minimal modern tasarıma sahip film arama uygulaması. OMDb API kullanarak film sonuçlarını poster kartları halinde listeler.

## Features
- Film arama
- Büyük poster kartları
- Loading state
- Error state
- Empty state
- Responsive layout

## Tech
- HTML
- CSS
- JavaScript
- OMDb API
